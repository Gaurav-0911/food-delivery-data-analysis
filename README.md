# Food Delivery Data Analysis

This project analyzes food delivery data by combining CSV, JSON, and SQL datasets.

## Files Used
- orders.csv (Transactional data)
- users.json (User master data)
- restaurants.sql (Restaurant master data)

## Tools & Technologies
- Python
- Pandas
- SQLite
- Google Colab

## Objective
To analyze user behavior, revenue trends, city-wise performance, cuisine analysis, and membership impact.

## Note
The final merged dataset is the single source of truth for all analysis.
